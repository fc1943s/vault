## 제칭그 스타르테드

### 인스탈라치옹

클로니 티 헤포지토리:

```포웨르제우
git clone https://github.com/i574n/vault.git
```

나비가치 투 티 지렉토리, 이니치알리지 데펜덴시스 안드 부이우드 티 프로젝트:

```포웨르제우
크드 바우우트
프즈 스크리프트스/워르크플로.프스1
```
